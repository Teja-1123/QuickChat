# 🚀 QuickChat

A modern real-time chat application with group chats, direct messaging, and file sharing — built using React, Node.js, MongoDB, and Socket.IO.

---

## ✨ Features

- 🔐 User Authentication (Login / Register)
- 💬 Real-time messaging (WebSockets)
- 👥 Create & join chat rooms
- 📩 Direct (1-to-1) messaging
- 📁 File sharing support
- ⚡ Fast and responsive UI (Vite + React)

---

## 🛠️ Tech Stack

- Frontend: React, Vite, CSS  
- Backend: Node.js, Express  
- Database: MongoDB  
- Real-time: Socket.IO

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

git clone https://github.com/Teja-1123/QuickChat.git  
cd QuickChat  

---

### 2️⃣ Setup Backend

cd server  
npm install  
npm run dev  

---

### 3️⃣ Setup Frontend

cd client  
npm install  
npm run dev  

---

## 🌐 Environment Variables

Create a .env file inside the server folder:

PORT=5000  
MONGO_URI=your_mongodb_connection_string  
JWT_SECRET=your_secret_key
